# Web-Design-Mini-project


**Saffron Table** is a modern Indian restaurant website developed as part of my second-semester Web Design course (B.Tech CSE, Semester 2).

### About the Project

This was my first complete website project. I started with a basic layout and iteratively improved it by studying real restaurant websites, analyzing their features, and refining the design. AI tools assisted with code generation, but the research, feature planning, and design decisions were my own.

### Features:

**Core Pages**
- Hero section with Book a Table and View Menu buttons
- Chef Story and Restaurant Story sections
- Visitor Information with location, hours, and an integrated map

**Interactive Menu**
- Categories: Snacks, Main Courses, Desserts, Beverages
- Filters: Veg / Non-Veg, Spicy, Gluten-Free
- Detailed items with pricing

**User Features**
- Mood-Based Dish Recommendations
- Table Reservation System
- Newsletter Subscription
- WhatsApp Support Chat
- Day / Night Theme Toggle

**Additional Sections**
- Featured Dishes showcase
- Guest Reviews
- Kitchen Notebook (food stories and cooking tips)
- Events & Catering (private dining, weddings, corporate, chef's table, and more)
- Online Ordering links (Zomato, Swiggy, Uber Eats)

### Tech Stack

HTML · CSS · JavaScript
